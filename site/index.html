WSBF CD Checkout

CpSc 372 Final Project Specification
David A. Cohen II

This site describes the development of a Python application to keep track of media check out to WSBF-FM DJs and interns. This application uses WSBF's existing MySQL database for authentication, album tracking, and checkout data storage.
