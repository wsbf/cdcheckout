Storyboard

Main Screen

The first screen will show a browsable list of albums to be reviewed. Users may select an album to check out by clicking "Check out" at any time, or type in the "Search" entry box to filter albums. Users may check out albums in the WSBF library (i.e. out of rotation) by clicking "Check Out From Library."

Checkout / Login

If no user is already logged in, the user is prompted to authenticate with either his/her wsbf.net credentials or (possibly in the future) Tiger1 card. After entering credentials, the user checks out the album by clicking "Check out album."

Success Dialog

After checking out the album, the user is presented with the choice to either continue checking out albums, or to log out. Either way, the main screen is shown again; however, if the user has chosen to continue browsing, s/he will not be prompted for her password.

Library Checkout

If the user is checking out an album from the library, s/he is brought to this screen which requires the album's artist and name. Here, a finite list of albums is implausible because the library is, for the most part, uncatalogued. Library checkouts require a signature from senior staff, so a member of senior staff must enter his/her credentials for approval.
